<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

Install the following globally : 
* go
  ```sh
  npm install npm@latest -g
  ```
* [wire]
  
* [make] 

### Setup

1. Clone the repo
   ```sh
   git clone https://github.com/shivamsk/go-blockchain.git
   
   ```
2. Install Go Dependencies
   ```sh
   go mod tidy
   ```
